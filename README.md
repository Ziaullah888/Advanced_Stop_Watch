#  Elegant Stopwatch

A beautiful, feature-rich stopwatch web application with lap timing, statistics tracking, and an elegant animated interface.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

##  Features

- **Precise Timing**: Track time with millisecond accuracy (hours:minutes:seconds:milliseconds)
- **Lap Timing**: Record multiple lap times while the stopwatch is running
- **Statistics Panel**: 
  - Best lap time tracking
  - Average lap time calculation
  - Total lap count
- **Lap History**: Scrollable list of all recorded lap times
- **Beautiful UI**: 
  - Animated gradient background
  - Floating particle effects
  - Glassmorphism design with backdrop blur
  - Smooth animations and transitions
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Modern Typography**: Uses Space Grotesk and Inter fonts from Google Fonts

##  Getting Started

### Prerequisites

No installation required! This is a pure HTML/CSS/JavaScript application that runs directly in your web browser.

### Installation

1. Clone the repository:
` ash
git clone https://github.com/ziaullah888/Advanced_Stop_Watch.git
` 

2. Navigate to the project directory:
` ash
cd Advanced_Stop_Watch
` 

3. Open index.html in your web browser:
   - Simply double-click the file, or
   - Right-click and select "Open with"  your preferred browser

##  Usage

1. **Start**: Click the "Start" button to begin timing
2. **Lap**: Once started, the Start button changes to "Lap" - click it to record lap times
3. **Stop**: Click "Stop" to pause the stopwatch
4. **Reset**: Click "Reset" to clear all times and lap history

### Features Explained

- **Lap Timing**: While the stopwatch is running, click "Lap" to record the current elapsed time without stopping the timer
- **Statistics**: The stats panel automatically updates to show:
  - Your fastest lap time
  - Average time across all laps
  - Total number of laps recorded
- **Lap History**: All recorded laps appear in reverse chronological order (newest first)
